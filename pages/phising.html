Phising

Phishing es una técnica utilizada por ciber delincuentes para obtener información confidencial de forma fraudulenta, como contraseñas, números de tarjeta de crédito, información bancaria, entre otros datos sensibles. Los ataques de phishing generalmente se realizan mediante correo electrónico, mensajes de texto o mensajes instantáneos, y suelen tener un aspecto legítimo, utilizando logos y diseños similares a los de empresas o entidades financieras conocidas. Estos mensajes engañan al usuario haciéndole creer que provienen de una fuente confiable y le solicitan que proporcione información personal. Una vez que el usuario proporciona la información solicitada, los ciber delincuentes pueden utilizarla para cometer fraudes, robar identidades o realizar transacciones no autorizadas.
